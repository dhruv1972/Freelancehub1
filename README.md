# FreelanceHub

A freelancing platform where clients can find freelancers for their projects.

## About

This is my web development project for connecting freelancers with clients. Users can sign up as either a freelancer or a client, browse projects, and manage their work.

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router

## How to Run

1. Install dependencies:
```
cd client
npm install
```

2. Start the dev server:
```
npm run dev
```

3. Open http://localhost:5173

## Features

- User registration and login
- Dashboard for users
- Project listings (coming soon)
- Messaging (coming soon)
